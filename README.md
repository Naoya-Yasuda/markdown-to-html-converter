# Markdown to HTML Converter

MarkdownファイルをHTMLに変換するCLIツールです。

## インストール

```bash
npm install
```

## ビルド

```bash
npm run build
```

## 使用方法

```bash
npm start <markdownファイルのパス>
```

例：
```bash
npm start ./example.md
```

## 機能

- MarkdownファイルをHTMLに変換
- カスタムテンプレートを使用したHTML生成
- シンプルで使いやすいCLIインターフェース
